# 자기소개 스크립트 (1~2분)

> 작성 기준: **결과물 중심** — 출시작과 프로젝트 성과를 전면에 내세워 '이런 결과물을 만든 사람'임을 강조

---

## 최종 스크립트

"구글 플레이 스토어에 정식 출시된 서스펜스 퍼즐 게임. 이 게임의 기획부터 개발, QA, 라이브 운영까지 전 사이클을 직접 책임진 이경민입니다.

창업 동아리 개발팀 팀장으로서 게임을 처음부터 끝까지 만들었습니다. 기획과 코딩에 참여하면서도 QA를 주도해 구글 플레이 정식 출시, 45회 다운로드와 2.1달러의 실제 광고 수익 창출까지 이어냈습니다. 출시 이후에도 유저 피드백을 기반으로 업데이트 품질을 검증하며 라이브 서비스의 전 과정을 몸으로 경험했습니다. 개발 중 발생한 카메라 오브젝트 중첩 오류는 로그 분석으로 원인을 추적해 해결했고, 이 경험이 구현 로직을 빠르게 읽어내는 분석력으로 이어졌습니다.

캡스톤 디자인에서는 전략 디펜스 게임의 QA를 맡아 장르에 맞는 시나리오 기반 테스트 케이스를 설계하고 버그 리포트를 체계화했습니다. Unity 기반 팀 프로젝트들을 통해 기획 의도와 구현 사이의 간극을 찾아내는 눈을 꾸준히 키워왔습니다.

게임을 직접 만들고, 출시하고, 유저 반응을 보며 품질을 다듬어온 경험. 이것이 저를 코드를 넘어 게임 전체를 이해하는 QA로 만들었다고 생각합니다. 게임 QA로서, 유저가 언제나 최고의 환경에서 플레이할 수 있도록 타이틀 품질을 완성해 내겠습니다. 감사합니다."

---

## 홍보 영상 구성안 (총 약 90초)

> 방향: **결과물 먼저 보여주고 → 만든 사람 소개** 순서. 말보다 화면이 먼저 임팩트를 준다.

---

### 장면 구성 요약

| # | 장면 | 시간 | 핵심 목적 |
|---|------|------|-----------|
| 1 | 훅 — 게임 플레이 | 0~8초 | "이 게임 뭐지?" 시선 끌기 |
| 2 | 자기소개 | 8~18초 | "이걸 만든 사람" 연결 |
| 3 | 출시 성과 | 18~42초 | 수치로 실력 증명 |
| 4 | 버그 해결 | 42~62초 | 분석력·QA 역량 구체화 |
| 5 | 캡스톤 프로젝트 | 62~76초 | QA 전문성 확장 |
| 6 | 마무리 | 76~90초 | 정체성 각인 + 포부 |

---

### 장면별 상세 구성 + 내레이션 대본

> PPT 캡처만으로 제작 가능 — 각 장면에 사용할 슬라이드 번호 명시

---

#### SCENE 1 · 훅 (0~8초)

**화면 소스: PPT 슬라이드 9 (WEIRD SPACE 홍보물 2장)**
- 홍보물 이미지 2장을 슬라이드쇼처럼 전환 (페이드 or 슬라이드 효과)
- BGM 시작 (긴장감 있는 로우 비트)
- 영상 끝에 텍스트 페이드인: `Google Play 정식 출시`

**내레이션** (없음 — 화면만으로 시선 확보)

---

#### SCENE 2 · 자기소개 (8~18초)

**화면 소스: PPT 슬라이드 8 (WEIRD SPACE 시작 화면 + 메인 화면 스크린샷)**
- 게임 화면 2장 배경으로 깔기 → 중앙에 이름 텍스트 등장
- 텍스트 오버레이: `이경민 · Game QA`

**내레이션**
> "기획부터 개발, QA, 라이브 운영까지.
> 이 게임의 전 사이클을 직접 책임졌습니다."

---

#### SCENE 3 · 출시 성과 (18~42초)

**화면 소스: PPT 슬라이드 9 (구글 플레이스토어 페이지 캡처)**
- 스토어 페이지 캡처 풀스크린
- 수치 텍스트 순서대로 등장: `누적 다운로드 45회` → `광고 수익 $2.1`
- 슬라이드 8 게임 설명·스테이지 화면도 컷으로 삽입

**내레이션**
> "창업 동아리 개발팀 팀장으로, 게임을 처음부터 끝까지 만들었습니다.
> 누적 다운로드 45회, 실제 광고 수익 2.1달러.
> 출시 이후에도 유저 피드백으로 업데이트 품질을 검증하며
> 라이브 서비스 전 과정을 직접 경험했습니다."

---

#### SCENE 4 · QA 역량 (42~62초)

**화면 소스: PPT 슬라이드 10 (WEIRD SPACE 테스트 케이스 + 버그리포트)**
- TC 문서 화면 → 버그리포트 화면 순서로 전환
- 텍스트 오버레이: `로그 분석 → 원인 추적 → 해결`

**내레이션**
> "개발 중 발생한 카메라 오브젝트 중첩 오류.
> 로그 분석으로 원인을 추적해 직접 해결했습니다.
> 이 경험이 구현 로직을 빠르게 읽어내는 분석력이 됐습니다."

---

#### SCENE 5 · 캡스톤 프로젝트 (62~76초)

**화면 소스: PPT 슬라이드 6 (RETURN OF THE DARK LORD 게임 화면 4장) + 슬라이드 7 (TC + 버그리포트)**
- 타이틀 화면 → 인게임 화면 컷 전환
- TC 문서 + 버그리포트 나란히 배치
- 텍스트 오버레이: `시나리오 기반 TC 설계 · 버그 리포트 체계화`

**내레이션**
> "캡스톤 디자인에서는 전략 디펜스 게임 QA를 전담.
> 장르에 특화된 테스트 케이스를 직접 설계하고
> 버그 리포트를 체계화했습니다."

---

#### SCENE 6 · 마무리 (76~90초)

**화면 소스: PPT 슬라이드 19 (감사 페이지) 참고 + 자체 텍스트 제작**
- 어두운 배경에 텍스트 순서대로 등장
  1. `만들고 · 출시하고 · 검증했습니다`
  2. `코드를 넘어, 게임 전체를 이해하는 QA`
  3. `이경민`
- BGM 페이드아웃

**내레이션**
> "게임을 직접 만들고, 출시하고, 유저 반응을 보며 품질을 다듬어온 사람.
> 코드를 넘어, 게임 전체를 이해하는 QA — 이경민입니다."

---

## 장면별 이미지 생성 프롬프트

> 각 프롬프트는 AI 이미지 생성기(Midjourney, DALL-E, Stable Diffusion 등)에 바로 입력 가능

---

#### SCENE 1 · 훅 — 서스펜스 분위기 배경 이미지


끝없이 이어지는 어두운 아파트 복도, 형광등이 미세하게 깜빡이는 불안한 조명, 복도 끝에 살짝 열린 문 하나, 차갑고 푸른 빛이 감도는 공간, 이상한 현상이 숨어 있을 것 같은 미스터리한 분위기, 시네마틱 구도, 게임 UI 없이 순수 배경 일러스트, 고해상도, 영화적인 색감
```

**분위기 키워드:** 불안감 · 미스터리 · 어두운 조명 · 퍼즐 게임 세계관

---

#### SCENE 2 · 자기소개 — 전문적인 타이틀 배경

```
딥 네이비 또는 다크 차콜 단색 배경, 중앙에 부드럽게 빛나는 얇은 선 그래픽 장식, 미니멀하고 현대적인 포트폴리오 타이틀 카드 디자인, 텍스트가 들어갈 여백이 충분한 구도, 차갑고 차분한 블루-그레이 컬러 팔레트, 게임 개발자 포트폴리오 인트로 느낌, 심플하고 세련된 분위기
```

**분위기 키워드:** 전문적 · 차분함 · 미니멀 · 신뢰감

---

#### SCENE 3 · 출시 성과 — 성취감 인포그래픽 배경

```
어두운 배경 위에 스마트폰 목업이 빛나는 구도, 화면 안에 앱 스토어 UI가 보이는 느낌, 금빛과 흰색 숫자 텍스트가 떠오르는 카운트업 애니메이션 스타일 일러스트, 성취와 출시를 상징하는 로켓 또는 체크마크 아이콘, 따뜻한 골드 포인트 컬러가 들어간 다크 배경, 인포그래픽 느낌의 깔끔한 레이아웃
```

**분위기 키워드:** 성취 · 출시 · 임팩트 있는 수치 · 골드 포인트

---

#### SCENE 4 · QA 역량 — 디버깅·분석 분위기

```
어두운 코드 에디터 화면이 배경으로 깔린 장면, 빨간색 오류 로그 메시지가 터미널 창에 표시된 모습, 화면 한쪽에 돋보기 또는 버그를 찾는 느낌의 아이콘, 초록빛 코드 텍스트가 흐르는 매트릭스 느낌의 배경, 분석과 문제 해결을 상징하는 기술적 분위기, 차갑고 어두운 화면 톤에 빨간색과 초록색 포인트
```

**분위기 키워드:** 분석 · 디버깅 · 기술적 · 문제 해결

---

#### SCENE 5 · 캡스톤 — 전략 디펜스 게임 세계관

```
다크 판타지 세계관의 성 방어 장면, 아이소메트릭 또는 탑다운 뷰 구도, 성벽 위에서 쏟아지는 마법 공격과 몰려오는 몬스터 군대, 보라색과 주황색의 대비되는 컬러 팔레트, 전략 게임 특유의 긴장감 있는 전투 장면, 다크 판타지 RPG 일러스트 스타일, 게임 기획 문서와 화면이 겹쳐 보이는 반투명 레이어 느낌
```

**분위기 키워드:** 다크 판타지 · 전략 · 긴장감 · 전투

---

#### SCENE 6 · 마무리 — 정체성 각인 클로징

```
완전한 검은 배경 위에 흰색 또는 은빛으로 빛나는 타이포그래피, 중앙에서 바깥으로 퍼져나가는 미세한 빛 효과, 게임 컨트롤러 실루엣과 돋보기 아이콘이 조화를 이루는 미니멀 아이콘, 포트폴리오 마지막 페이지처럼 여백이 넉넉하고 차분한 구도, 어둠 속에서 한 줄기 빛이 비치는 시네마틱한 분위기, 클로징 크레딧 느낌
```

**분위기 키워드:** 각인 · 여운 · 미니멀 · 시네마틱
